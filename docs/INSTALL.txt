vindula.joomlamigration Installation
------------------------------------

To install vindula.joomlamigration using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``vindula.joomlamigration`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        vindula.joomlamigration
       
* Re-run buildout, e.g. with:

    $ ./bin/buildout
